Overgrown

Project Goal:

To turn the Lego like game Minecraft into a procedurally generated derelict cityscape, provide a detailed narrative adventure for the player through unique resource acquisition, quests, and NPCs, as well as compiling all required files to be run on server hardware for multiplayer support. Once the custom world gen, progression, and hosting compatibility was complete, I would need to advertise and network with other creators to bring attention to the project and continue to support the modpack as a whole with bug support and updates.

Creation Process:

Minecraft Forge is a modification API that allows for bridging custom content into the base game, and Amazons Twitch streaming program is a widely accepted hosting platform for modifications. By using a series of mods I created, and other popular community created staples, I was able to turn Minecraft into an infinite city apocalypse generation tool. From there I wrote up a short lore for the modpack and a compelling questline centered around rebuilding machines and farming from junk scavenged in the ruins. Once the pack was released, I worked closely with Youtubers and Twitch Livetreamers to promote the pack, where this project became the 25th most downloaded pack ever (out of 3020 for that version of the game), at 168,202 unique downloads.
